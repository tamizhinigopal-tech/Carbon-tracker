## 🌍 Carbon Emission Tracker

## 📌 Project Overview

The Carbon Emission Tracker is a full-stack web application that helps estimate carbon emissions based on industrial inputs like diesel usage and electricity consumption. It also provides risk analysis and smart suggestions to reduce emissions.

---

## 🚀 Features

* Calculate total carbon emissions
* Separate emission values for diesel and electricity
* Risk level classification (Low / Medium / High)
* Smart suggestions for emission reduction
* Interactive pie chart visualization
* Clean and responsive user interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Libraries:** Chart.js
* **Deployment:** Render

## ⚙️ How It Works

1. User enters mine details (diesel, electricity, etc.)
2. Data is sent to backend using API
3. Backend calculates emissions
4. Results are displayed on dashboard with charts and suggestions
